# Employee Absence Planner

A comprehensive application for managing employee timehseets, approving timesheet and payment processing.

## Features

- Track employee timesheet
- Manage approvals and rejection of the timesheets
- Processing timesheet for billing
- Integration with HR systems
- Reporting and analytics

## Technology Stack

- SAP CAP (Cloud Application Programming Model)
- SAPUI5 / Fiori Elements
- SAP HANA Cloud Database
- Node.js Runtime

## Getting Started

```bash
npm install
npm run dev
```
